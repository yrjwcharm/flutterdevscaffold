```javascript
var callbacks = [];
for (var i = 0; i < 2; i++) {
  callbacks.push(() => console.log(i));
}
for (var c of callbacks) {
  c();
}
```

- 输出 2,2

```dart
var callbacks = [];
  for (var i = 0; i < 2; i++) {
    callbacks.add(() => print(i));
  }

  for (final c in callbacks) {
    c();
  }
```

- 输出 0,1

一句话总结
var：只有函数作用域，整个 for 循环共享同一个 i，闭包保存的是变量引用，循环结束后 i = 2，所以打印 2 2。
let：具有块级作用域，并且在 for 循环中每次迭代都会创建新的变量绑定，因此每个闭包都会保存各自那一轮的 i，打印 0 1。

面试中常见的一句话：闭包捕获的不是变量的值，而是变量本身（准确地说，是变量绑定）。 var 只有一个绑定，而 let 在 for 循环中每次迭代都会产生一个新的绑定。

```javascript
let num = 5;
switch (num) {
  case 1:
    console.log(1);
  case 2:
    console.log(2);
  case 5:
    console.log(3);
  case 4:
    console.log(4);
  default:
    console.log("default");
}
```

### Flutter UI 适配安全区域 如果使用系统AppBar组件，它自动会适配顶部安全区域,

```dart
import 'package:flutter/material.dart';

void main() {
  runApp(
    MaterialApp(
      home: Scaffold(
         builder: (context, child) {
        final top = MediaQuery.of(context).padding.top;
        print('top: $top');
        return child!;
      },
        appBar: AppBar(
          title: const Text('PreferredSize Example'),
          titleTextStyle: TextStyle(fontSize: 20, color: Colors.white),
          centerTitle: false,
          backgroundColor: Colors.blue,
        ),
        body: SafeArea(top: false, child: const AppWidget()),
      ),
    ),
  );
}

class AppWidget extends StatefulWidget {
  const AppWidget({super.key});

  @override
  State<AppWidget> createState() => _AppWidgetState();
}

class _AppWidgetState extends State<AppWidget> {
  @override
  Widget build(BuildContext context) {
    return Container(
      color: Colors.red,
      width: double.infinity,
      height: double.infinity,
      child: const Center(
        child: Text(
          'Hello, World!',
          style: TextStyle(fontSize: 24, color: Colors.white),
        ),
      ),
    );
  }
}

```
